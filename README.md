# Music Box

## install

### python3 virtualenv

```bash
$ cd
$ python3 -m venv env1
$ cd env1
$ . bin/activate
```

### clone git repository

```bash
$ cd ~/env1

$ git clone git@github.com:ytani01/music-box.git
or 
$ git clone https://github.com/ytani01/music-box.git
```

### setup.sh

```bash
$ cd ~/env1/music-box
$ . ../bin/activate
$ ./setup.sh
```
